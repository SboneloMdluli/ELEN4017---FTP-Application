ELEN4017 : FTP SERVER - CLIENT FILE TRANSFER APPLICATION USING SOCKETS

The application is programmed using python 3.6 and on Ubuntu 20.04.4 LTS.
Basic functionality is implemented following the RFC959 standards. 
The presentation contains a video of the set up and file structure and how to use the code.

Before using the application you may want to:

1. allow firewall to allow access to privileged ports with UFW firewall 
(https://www.digitalocean.com/community/tutorials/how-to-set-up-a-firewall-with-ufw-on-ubuntu-18-04)
2. Set up a python environment 
(https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/)


SERVER
*The server can be started from the terminal using

$ sudo python server.py


CLIENT
*The client can be started from the terminal using

$ sudo python3 cc.py
